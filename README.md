# Portfolio (GitHub Pages)

Dit is een simpele portfolio website met meerdere pagina's:
- Home
- About
- Resume
- Projects
- Contact

## Hoe publiceren via GitHub Pages

1. Zet deze bestanden in je repository: `<username>.github.io`
2. Ga naar **Settings** van de repo
3. Ga naar **Pages**
4. Bij **Build and deployment**:
   - Source: **Deploy from a branch**
   - Branch: **main** (of master) en **/(root)**
5. Wacht even en open:
   `https://<username>.github.io`

## Aanpassen

- Tekst aanpassen: open de `.html` bestanden
- Styling: `assets/css/style.css`
- JS: `assets/js/main.js`

Veel succes!
